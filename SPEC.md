# 飞机大战网页游戏规格文档

## 1. 项目概述
- **项目名称**: Space Shooter
- **项目类型**: 2D 街机射击游戏
- **核心功能**: 玩家控制飞船射击敌人，获取分数，躲避敌人子弹
- **目标用户**: 休闲游戏玩家

## 2. UI/UX 规格

### 布局结构
- 游戏画布: 800x600 像素，居中显示
- 顶部状态栏: 显示分数和生命值
- 游戏区域: 全屏深空背景

### 视觉设计
- **色彩方案**:
  - 背景: #0a0a1a (深空蓝黑)
  - 玩家飞机: #00ff88 (霓虹绿)
  - 敌人: #ff3366 (霓虹红)
  - 子弹-玩家: #00ffff (青色)
  - 子弹-敌人: #ff6600 (橙色)
  - 星星: #ffffff (白色)
  - UI文字: #ffffff
- **字体**: "Orbitron", sans-serif (科幻风格)
- **视觉效果**:
  - 星空背景缓慢移动
  - 爆炸粒子效果
  - 飞机引擎尾焰
  - 击中敌人时闪光

### 组件
- 玩家飞机: 三角形飞船造型
- 敌人: 红色菱形飞船
- 子弹: 细长矩形
- 星星: 随机大小亮点
- 开始/结束界面: 半透明遮罩 + 文字

## 3. 功能规格

### 核心功能
1. **玩家控制**
   - 方向键或WASD移动飞机
   - 空格键或鼠标点击发射子弹
   - 飞机不能移出画布边界

2. **敌人系统**
   - 敌人从顶部随机位置生成
   - 敌人向下移动并略微左右摆动
   - 敌人随机发射子弹

3. **碰撞检测**
   - 玩家子弹击中敌人: 敌人死亡，玩家得分
   - 敌人子弹击中玩家: 玩家失去生命
   - 玩家撞击敌人: 玩家失去生命

4. **游戏状态**
   - 3条生命
   - 分数系统: 每个敌人100分
   - 游戏结束显示最终分数
   - 点击重新开始

### 用户交互流程
1. 加载游戏 → 显示开始界面
2. 点击开始 → 游戏进行
3. 生命归零 → 游戏结束，显示分数
4. 点击重新开始 → 回到步骤2
