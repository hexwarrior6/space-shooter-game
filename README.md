# Space Shooter

一个 2D 街机射击游戏，用于测试 Claude Code 连接 Minimax 2.5 模型。

## 游戏介绍

这是一款经典的飞机大战游戏，玩家控制飞船在太空中与敌人作战。

### 玩法

- **移动**: 方向键 或 WASD
- **射击**: 空格键 或 鼠标点击
- **目标**: 消灭敌人获得分数，躲避敌人子弹
- **生命**: 3 条生命，撞到敌人或被击中会失去生命
- **得分**: 每消灭一个敌人 +100 分

## 技术栈

- 纯 HTML5 Canvas 实现
- 无需任何依赖，直接在浏览器打开 `index.html` 即可运行

## 运行

直接在浏览器中打开 `index.html` 文件即可开始游戏。
